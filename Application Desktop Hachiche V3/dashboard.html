Bienvenue sur le Dashboard

Fonctionnement de l'application

Cette application vous permet de gérer le stock de cannabis de manière efficace et organisée. Voici une description des différentes sections de l'application :

Notre Stock 🗃️

Cette section vous offre une vue d'ensemble de notre stock actuel, incluant les quantités disponibles et les informations essentielles à connaître.

Responsable Décontamination 🌿

Ici, vous pouvez gérer les responsables de décontamination, assigner des tâches et suivre les progrès.

Entreposage 📦

Cette section vous permet de gérer l'entreposage des plants, y compris l'ajout, le suivi et l'organisation des stocks.

Plantules 🌱

Dans cette section, vous pouvez gérer les plantules, y compris leur ajout, modification, et suppression.

Modifications 🛠️

Cette section permet de modifier les informations existantes sur les plants et autres éléments de l'inventaire.

Upload 📁

Cette section vous permet de télécharger des fichiers pour mettre à jour l'inventaire en masse.

Déconnexion 🚪

Utilisez ce lien pour vous déconnecter de l'application de manière sécurisée.
